# 實驗3: ASU Only vs ASU+MSU 比較

## 研究問題
探討 Market Style Unit (MSU) 對不同規模股票池的影響。MSU 輸出 rho 值控制多空配置比例，預期能透過市場擇時提升績效。

## 實驗配置

| 符號 | 說明 | 模型架構 | MSU 輸入 |
|------|------|----------|----------|
| A_47 / A_22 | Baseline | ASU Only | - |
| A_47_M_O / A_22_M_O | +MSU (Original) | ASU+MSU | OHLCV |
| A_47_M_S / A_22_M_S | +MSU (Sentiment) | ASU+MSU | Sentiment (待補) |

**實驗期間**：訓練 2016-2020 (5年) / 驗證 2021-2023 (3年) / 測試 2024-2025 (2年)

## 總結

1. **47支：MSU 有效**
   - ASR +5.6%、ARR +7.86pp、勝率 +10pp
   - rho 幅度較大（max 達 0.44），能積極調整多空比例實現擇時增益

2. **22支：MSU 無效**
   - ASR -7.9%、ARR -4.46pp、勝率 -13pp
   - MSU 未學到有效信號，rho 值偏小（max 0.14 vs 47支的 0.44），無法脫離多空對做

3. **關鍵發現**
   - MSU 需要足夠的股票池規模才能學習有效的市場風格信號
   - 有效的 MSU 會在市場好時輸出較大 rho，脫離多空對做、積極做多
   - 小型股票池的 MSU 訓練不穩定，無法學到有效信號，不建議啟用

4. **未來方向**
