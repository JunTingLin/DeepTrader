# 實驗3: 新聞特徵效益分析（Sentiment / Embedding）

## 研究問題
評估新聞情緒分數 (Sentiment) 與新聞嵌入融合 (Embedding) 對模型表現的影響。假設：47支股票池中有 25 支新聞覆蓋率極低（零向量），導致新聞特徵成為雜訊；而 22支高覆蓋率股票池應能充分發揮新聞特徵效益。

## 實驗配置

| 符號 | 說明 | 架構 |
|------|------|------|
| A_47 | 47支基準 | ASU |
| A_47_S | 47支+情緒 | ASU+MSU+Sent |
| A_47_E | 47支+嵌入 | ASU+MSU+Emb |
| A_47_SE | 47支+情緒+嵌入 | ASU+MSU+Sent+Emb |
| A_22 | 22支基準 | ASU |
| A_22_S | 22支+情緒 | ASU+MSU+Sent |
| A_22_E | 22支+嵌入 | ASU+MSU+Emb |
| A_22_SE | 22支+情緒+嵌入 | ASU+MSU+Sent+Emb |

**實驗期間**：訓練 2016-2020 (5年) / 驗證 2021-2023 (3年) / 測試 2024-2025 (2年)

## 總結

1. **47支股票池：新聞特徵效益有限**
   - 25 支低覆蓋率股票的零向量成為雜訊
   - Embedding 反而拖累表現（Test ASR 1.24 vs 基準 1.50）
   - 所有變體皆落後 0050

2. **22支股票池：新聞特徵效益顯著**
   - **A_22_SE 達到 Test ASR 2.22（+35%）、ARR 61.45%（+25pp）**
   - 大幅超越 0050（ASR 1.82）
   - 勝率全面領先（Per Trade 65%、3M 76%、6ME 67%）

3. **Sentiment vs Embedding**
   - 單獨使用 Sentiment 效果不佳
   - Embedding 融合在高覆蓋率股票池中有效
   - 兩者結合 (Sent+Emb) 達到最佳效果

4. **建議**
   - 採用 22支高新聞覆蓋率股票池
   - 使用完整架構 (ASU+MSU+Sent+Emb)
   - 後續可探索提升低覆蓋率股票的新聞特徵品質
