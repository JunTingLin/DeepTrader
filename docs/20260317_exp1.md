| 參數 | 47支 (A_47) | 22支 (A_22) |
|------|-------------|-------------|
| 模型架構 | ASU Only (msu_bool=False) | ASU Only (msu_bool=False) |
| 成分股來源 | 完整股票池 | 新聞高覆蓋子集 (覆蓋率 >60%) |

## 實驗1: 實驗結果 - Risk-Return

### 核心發現

**47支組合在風險調整後報酬率上顯著優於22支組合，且兩者皆大幅超越 0050 基準。**

#### 關鍵數據對比

| 期間 | 指標 | A_47 | A_22 | 差異 | 0050 |
|------|------|------|------|------|------|
| **Validation** | ASR | **1.77** | 1.55 | -12.4% | 0.39 |
| | ARR | **49.88%** | 32.47% | -17.4pp | 7.97% |
| | MDD | **7.73%** | 21.16% | +13.4pp | 31.56% |
| | AVOL | 28.32% | **21.02%** | -7.3pp | 20.34% |
| **Test** | ASR | **2.32** | 1.77 | -23.7% | 1.82 |
| | ARR | **54.96%** | 43.74% | -11.2pp | 39.28% |
| | MDD | **11.73%** | 12.51% | +0.8pp | 14.37% |
| | AVOL | **23.79%** | 24.70% | +0.9pp | 21.61% |

### 分析與解釋

#### 1. 夏普比率 (ASR) 差異分析
- **Test Period**: A_47 的 ASR (2.32) 比 A_22 (1.77) 高出 **31%**
- **vs 0050**: A_47 超越 0050 達 **27.7%**，而 A_22 僅與 0050 持平 (1.77 vs 1.82)
- **意義**：47支組合在每單位風險下獲得更高報酬

#### 2. 最大回撤 (MDD) 差異分析
- **Test Period**: A_47 的 MDD (11.73%) 略低於 A_22 (12.51%)，差異 0.78pp
- **關鍵洞察**：兩者 MDD 接近且皆優於 0050 (14.37%)，顯示模型風險控制有效

#### 3. 報酬率 (ARR) 差異原因
A_47 年化報酬率較高的可能原因：
1. **更廣泛的 Alpha 機會**：47支股票提供更多潛在獲利機會
2. **多空標的更豐富**：更多可配置標的供模型做多或做空
3. **TE 學習效益**：Temporal Encoder 在更多資產上學習，提供更豐富的時序特徵表達

#### 4. 波動率與報酬分析
- **Test Period**: A_47 波動率 (23.79%) 略低於 A_22 (24.70%)，且報酬率更高
- **意義**：A_47 實現「低風險、高報酬」，風險調整後表現全面優於 A_22

## 實驗1: 實驗結果 - Win-Rate

### 核心發現

**47支組合在驗證期的短中期勝率顯著較高，但22支組合在測試期的單筆交易勝率反而更高 — 這揭示了「勝率」與「獲利能力」的關鍵差異。**

#### 關鍵數據對比

| 期間 | 勝率類型 | A_47 | A_22 | 差異 |
|------|----------|------|------|------|
| **Validation** | Per Trade | **70.59%** | 57.84% | +12.75pp |
| | 3 Months | **81.82%** | 63.64% | +18.18pp |
| | 6 Months | 66.67% | **80.00%** | -13.33pp |
| **Test** | Per Trade | 47.83% | **59.42%** | -11.59pp |
| | 3 Months | **66.67%** | 57.14% | +9.53pp |
| | 6 Months | 66.67% | 66.67% | 0pp |

### 分析與解釋

#### 1. 「勝率悖論」現象
測試期出現有趣現象：**A_22 單筆交易勝率 (59.42%) 高於 A_47 (47.83%)，但 ARR 反而較低 (43.74% vs 54.96%)**

**解讀**：
- A_22 傾向於「小贏多次」— 勝率高但單筆獲利幅度小
- A_47 傾向於「大贏少次」— 勝率略低但單筆獲利幅度大
- **投資含義**：追求高勝率不一定帶來高報酬，報酬的「幅度」同樣重要

#### 2. 時間尺度效應
隨著評估時間拉長 (3個月、6個月)，A_47 的優勢逐漸顯現：
- 短期波動被過濾，模型的 alpha 能力更明顯
- A_47 在季度評估下維持 66.67% 勝率，顯示**穩定的中長期投資能力**

#### 3. Validation vs Test 的勝率下降
兩組模型在測試期的單筆勝率都明顯下降：
- A_47: 70.59% → 47.83% (-22.76pp)
- A_22: 57.84% → 59.42% (+1.58pp)

**可能原因**：
- 測試期 (2024-01 ~ 2025-12) 涵蓋 AI 概念股急漲與市場高波動，與訓練期市場結構差異大
- A_47 因涵蓋更多股票，更容易受市場風格輪動影響
- 但 A_47 仍維持較高 ARR，說明其「輸小贏大」的策略仍然有效

#### 4. 實務建議
- 若追求**心理舒適度**（較少虧損交易）：可考慮 A_22
- 若追求**最終報酬最大化**：應選擇 A_47
- 機構投資人通常偏好 A_47 類型策略

## 實驗1: 實驗結果 - Accuracy

### 核心發現

**47支組合在選股精準度 (Precision) 上顯著領先，顯示模型在更大股票池中仍能精準挑選上漲標的。**

#### 關鍵數據對比

| 期間 | 指標 | A_47 | A_22 | 差異 |
|------|------|------|------|------|
| **Validation** | Long P@4 | **66.91%** | 59.31% | +7.60pp |
| | Short P@4 | **50.49%** | 45.10% | +5.39pp |
| **Test** | Long P@4 | **66.30%** | 52.17% | +14.13pp |
| | Short P@4 | **50.00%** | 44.93% | +5.07pp |

> **註**：Recall@4 不列入比較，因 22 支的隨機命中率 (4/22=18.2%) 本就高於 47 支 (4/47=8.5%)，跨池比較不公平。

### 分析與解釋

#### 1. Precision 定義與洞察

**Precision@4**：模型選出做多/做空的 4 支股票中，實際上漲/下跌的比率

**關鍵洞察**：A_47 的 Long Precision (66.30%) 表示模型做多的4支股票中有約2/3確實上漲，在更大股票池中仍維持高準確度，顯示模型**真正具備選股能力**。

#### 2. 做多 vs 做空能力分析

| 方向 | A_47 Test P@4 | A_22 Test P@4 | 觀察 |
|------|---------------|---------------|------|
| Long | 66.30% | 52.17% | A_47 優勢明顯 (+14.13pp) |
| Short | 50.00% | 44.93% | A_47 小幅領先 (+5.07pp) |

**發現**：
- 兩組模型的做多能力都顯著優於做空能力
- 這可能反映台股市場的**多頭偏向**特性，或模型在識別下跌股票上的限制
- A_47 的做空精準度 (50%) 已達隨機水準，未來可作為優化方向

#### 3. Precision 穩定性分析

| Model | Validation Long P@4 | Test Long P@4 | 變化 |
|-------|---------------------|---------------|------|
| A_47 | 66.91% | 66.30% | -0.61pp |
| A_22 | 59.31% | 52.17% | -7.14pp |

**關鍵發現**：A_47 的 Precision 在驗證期與測試期幾乎一致，顯示**模型泛化能力優異**，而 A_22 出現明顯下降 (-7.14 百分點)，可能存在過擬合風險。

## 實驗1: 總結

### 主要結論

#### 1. 47支組合全面優於22支組合

| 面向 | A_47 優勢 | 統計證據 |
|------|----------|----------|
| **風險調整報酬** | 顯著較高 | Test ASR: 2.32 vs 1.77 (+31%) |
| **絕對報酬** | 顯著較高 | Test ARR: 54.96% vs 43.74% (+11.22pp) |
| **風險控制** | 最大回撤更低 | Val MDD: 7.73% vs 21.16% |
| **選股精準度** | Precision 更高 | Test Long P@4: 66.30% vs 52.17% |
| **模型穩定性** | 泛化能力更強 | Precision 變化: -0.61pp vs -7.14pp |

#### 2. 唯一例外：單筆交易勝率
A_22 在測試期單筆勝率 (59.42%) 高於 A_47 (47.83%)，但這**不影響最終獲利能力**：
- 高勝率 ≠ 高報酬
- A_47 屬於「輸小贏大」型策略，更適合專業投資

#### 3. 47支優勢的根本原因（純價格模型）

| 原因 | 說明 |
|------|------|
| **分散化效益** | 更多股票 = 更低的非系統性風險 = 更低 MDD |
| **Alpha 機會** | 更大的選股池 = 更多潛在獲利機會 |
| **TE 學習效益** | 更多資產的時序特徵 = 更強的 Transformer 學習信號 |

#### 4. 承上啟下：實驗2預告

本實驗建立了純價格模型的 Baseline：**47支 > 22支**

但這引發一個關鍵問題：
> **若納入新聞與情感資訊，結果是否會改變？**

實驗2將探討：
- 47支中有 25 支新聞覆蓋率低，納入新聞可能引入 **雜訊 (Noise)**
- 22支全部為高新聞覆蓋率，新聞訊號品質較高
- 預期結果：**22支 + 新聞 可能反超 47支 + 新聞**
